## Rikybot
Whatsapp Bot For Termux

## CARA INSTALL
```bash
> apt update && apt upgrade
> apt install git
> apt install ffmpeg
> apt install nodejs
> apt install wget
> git clone https://github.com/Riky2094/Rikybot
> cd Rikybot
> bash install.sh
> node index.js
```
